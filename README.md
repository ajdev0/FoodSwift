# FoodSwift

A food delivery application made using React and Swiggy's API.

## Tech Stack

- React
- Parcel
- Tailwind CSS
- Jest(Testing)
- Redux-toolkit
- Auth0

## Features

- Get restaurants
- Add to cart
- Shimmer UI

## How to use

1. Clone the repo

```bash
git clone https://github.com/ajdev0/FoodSwift
```

2. Install dependencies

```bash
npm install
```

3. Start the dev server

```bash
npm start
```
